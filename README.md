
# Cinike

Discord Bot


## Features

- mass delete
- say


## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

`DISCORD_TOKEN`


## Run Locally

Clone the project

```bash
  git clone https://github.com/pramaaa/Cinike.git
```

Go to the project directory

```bash
  cd Cinike
```

Start

```bash
  python.exe main.py
```
